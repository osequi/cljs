# Cheatsheet

## Syntax

| Characters | Meaning                                     |
| ---------- | ------------------------------------------- |
| **\x**     | Character                                   |
| **#""**    | Regexp                                      |
| **:xxx**   | Keyword                                     |
| **'**      | Delayed evaluation                          |
| **()**     | List                                        |
| **[]**     | Vector                                      |
| **#{}**    | Set                                         |
| **{}**     | Map                                         |
| **&**      | The rest of params for a variadic function  |
| **#()**    | Anonymous function                          |
| **%**      | The default param for an anonymous function |
| **%1**     | The first param for an anonymous function   |
